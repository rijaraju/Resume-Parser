# Resume-Parser

## Description

Create a resume parser to extract Key information from the resumes the input data is given in JSON format with the following fields tagged :/
      a)	Location/
      b)	Designation/
      c)	Name/
      d)	Years of Experience/
      e)	College/
      f)	Degree/
      g)	Graduation Year/
      h)	Companies worked at/
      i)	Email address/
