# Resume-Parser

## Description

Create a resume parser using Spacy to extract key information from the resumes./n The input data is given in JSON format with the following fields tagged :\
      a)	Location\
      b)	Designation\
      c)	Name\
      d)	Years of Experience\
      e)	College\
      f)	Degree\
      g)	Graduation Year\
      h)	Companies worked at\
      i)	Email address
